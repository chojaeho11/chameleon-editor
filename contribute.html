진행 방법

STEP 1

업로드

위 카드에서 유형 선택 → 파일과 이름 입력 → 검토 요청.

STEP 2

관리자 검토

저작권 · 품질 검토 후 승인 또는 거절. 거절 시 row 자동 삭제.

STEP 3

적립 + 판매 보너스

승인 즉시 기본 적립금 지급. 사용자가 내 디자인으로 주문하면 결제 금액의 3%가 자동 추가 적립.

자주 묻는 질문

저작권은 어떻게 처리되나요?

본인이 직접 만든 디자인만 업로드 가능합니다. 무단 도용·복제 의심 시 거절되며, 발견 즉시 해당 디자인은 비공개 처리 후 적립금이 회수됩니다.

적립금은 어떻게 사용/환전하나요?

My page → 마일리지에서 확인 가능. 사이트 내 주문 결제에 사용하거나, 일정 금액 이상부터 현금 환전 신청이 가능합니다 (KR 사이트 기준).

3% 판매 보너스는 어떻게 계산되나요?

고객이 내 디자인(템플릿/벡터/PNG/로고)을 적용해서 주문을 완료하면, 그 주문의 결제 금액 × 3% 가 자동으로 내 마일리지에 추가됩니다. 환불 시에는 차감됩니다.

디자이너가 아닌데 가능한가요?

네! 카멜레온 에디터 안에 사진·벡터·요소·장식이 이미 많이 들어 있습니다. 그것들을 이리저리 조합해서 감각있게 배치하면 그 자체로 훌륭한 템플릿이 됩니다. 그림 실력보다 색감·배치·아이디어가 중요합니다.
