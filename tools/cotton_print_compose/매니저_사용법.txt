  📌 Cotton Print 인쇄 데이터 생성기 - 매니저 사용법
═══════════════════════════════════════════════════════

작업지시서를 직접 손으로 합성할 필요 없이, 주문번호만 입력하면
자동으로 인쇄 데이터(TIFF/PNG/JPG) 가 생성됩니다.


【 처음 한 번만 — 설치 】

▶ Windows 사용자

  1. 카톡으로 받은  Cotton_Print_생성.bat  파일을 바탕화면에 저장
     (또는 어디든 원하는 폴더)

  2. 더블클릭으로 실행

  3. Python 이 없으면 자동으로 다운로드 페이지가 열림
     → "Download Python 3.x" 클릭
     → 설치 시 "Add Python to PATH" 체크박스 반드시 체크
     → 설치 완료 후 다시 BAT 더블클릭

  4. 처음 실행시 Pillow 라이브러리가 자동 설치됨 (1-2분)

▶ Mac 사용자

  1. 카톡으로 받은  Cotton_Print_생성_Mac.command  파일 저장

  2. 처음에는 더블클릭이 안 될 수 있음 → 우클릭 → "열기"
     → 보안 경고 뜨면 "열기" 클릭 (한 번만)

  3. 자동으로 필요한 것 모두 설치됨


【 매번 사용 】

  1. Cotton_Print_생성  파일 더블클릭

  2. 주문번호 입력 (예: 3553)
     → 작업지시서 페이지 URL 의 ?id= 뒤 숫자

  3. 포맷 선택 (그냥 Enter 누르면 TIFF)

  4. 5-30초 후 자동으로 output 폴더가 열림
     → 그 안에 TIFF 파일이 들어있음

  5. TIFF 파일을 인쇄소에 전달


【 자주 묻는 질문 】

Q. "artwork_url 없음" 메시지가 나와요
A. 작업지시서에 디자인 파일이 안 올라간 상태입니다.
   먼저 작업지시서 페이지에서 보라색 "파일 첨부" 또는 "파일 교체" 로
   디자인 파일을 업로드한 후 다시 실행하세요.

Q. 파일이 너무 커요
A. 다음 실행시 포맷 선택에서 "2" (JPG) 선택 → 파일 크기 1/5

Q. 인쇄 화질이 부족해요
A. 다음 실행시 옵션 입력 화면에서 "--dpi 300" 추가
   → 화질 2배, 파일 4배 커짐

Q. 실행하니 검은 창만 뜨고 사라져요
A. Python 이 설치 안 됐거나 PATH 설정이 안 된 상태입니다.
   본사 IT 담당자에게 문의.

Q. 결과물 폴더는 어디에 생기나요?
A. BAT/Command 파일이 있는 폴더의 output\ 안에 생성됩니다.
