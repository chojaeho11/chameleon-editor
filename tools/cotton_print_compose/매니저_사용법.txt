  📌 Cotton Print 인쇄 데이터 생성기 - 매니저 사용법
═══════════════════════════════════════════════════════

작업지시서 주문번호만 입력하면 인쇄 데이터(TIFF) 가
자동으로 만들어집니다. 마우스 클릭만으로 끝!


【 설치 — 처음 한 번만 】

▶ 가장 쉬운 방법 (강력 추천)

  1. 카톡으로 받은  Cotton_Print_설치.bat  를 더블클릭

  2. 자동으로 다음이 실행됨:
     ✅ Python 설치 확인 (없으면 다운로드 페이지 자동 오픈)
     ✅ Desktop\Cotton_Print 폴더 생성
     ✅ 필요한 파일들 자동 다운로드
     ✅ Pillow 라이브러리 자동 설치
     ✅ 바탕화면에 "Cotton Print" 바로가기 생성
     ✅ GUI 자동 실행

  3. 끝. 다음부터는 바탕화면의 "Cotton Print" 더블클릭

▶ Python 이 없는 경우

  설치 BAT 가 자동으로 다운로드 페이지를 엽니다.
  설치 시 "Add Python to PATH" 체크박스 반드시 체크 후
  설치 완료되면 BAT 다시 더블클릭.


【 매번 사용 】

  1. 바탕화면의 "Cotton Print" 바로가기 더블클릭
     (또는 설치 폴더의 Cotton_Print_생성.bat)

  2. GUI 화면이 열림:
     ┌────────────────────────────────────┐
     │ 🖨️ Cotton Print                    │
     │ 패브릭 인쇄 데이터 생성기           │
     ├────────────────────────────────────┤
     │                                      │
     │ 주문번호: [_____________]            │
     │                                      │
     │ 출력 포맷: ◉ TIFF (인쇄소 표준)      │
     │            ○ JPG (용량 작음)         │
     │            ○ PNG (투명 배경)         │
     │                                      │
     │ 항목 선택: ☑ 모든 항목               │
     │                                      │
     │ 해상도: [150 ▼] dpi                  │
     │                                      │
     │ [   🎨 인쇄 데이터 생성   ]          │
     │                                      │
     │ ── 실행 로그 ──                       │
     │ [..] 다운로드 중...                  │
     │ [OK] 합성 중...                      │
     │ ✅ 완료!                              │
     │                                      │
     │ [output 열기] [도움말] [🔄 업데이트] │
     └────────────────────────────────────┘

  3. 주문번호 입력 → [인쇄 데이터 생성] 버튼 클릭

  4. 5-30초 후 output 폴더가 자동으로 열림

  5. TIFF 파일을 인쇄소에 전달


【 업그레이드 — 어떻게 되나요? 】

  ✨ 완전 자동입니다.

  매번 BAT 더블클릭할 때마다 자동으로 최신 버전이
  다운로드됩니다. 매니저는 아무것도 안 하셔도 됩니다.

  [동작 원리]
  - BAT 시작시 → Cloudflare 에서 gui.py / compose_fabric.py
    최신 버전 자동 다운로드 → GUI 실행
  - 인터넷이 안 되면 기존 버전 그대로 사용 (오프라인 대응)

  [수동 업데이트]
  - GUI 우측 하단 [🔄 업데이트 확인] 버튼 클릭
  - 즉시 최신 버전 확인 + 다운로드
  - 새 버전 있으면 재시작 안내


【 자주 묻는 질문 】

Q. "artwork_url 없음" 에러
A. 작업지시서에 디자인 파일이 안 올라간 상태입니다.
   매니저 작업지시서 페이지에서 보라색 [파일 첨부] 또는
   [파일 교체] 로 업로드 후 다시 실행.

Q. 파일이 너무 커요
A. 출력 포맷에서 "JPG" 선택 → 용량 1/5

Q. 인쇄 화질이 부족해요
A. 해상도에서 "300" 선택 → 화질 2배, 파일 4배 커짐

Q. GUI 가 안 열려요
A. Python 이 설치 안 된 상태.
   본사 IT 담당자 또는 본사로 문의.

Q. 처음 실행 시 시간이 오래 걸려요
A. Pillow 라이브러리 설치 + 스크립트 다운로드 중입니다 (1-2분).
   GUI 안의 로그 창에서 진행 상황이 보입니다.

Q. 결과물 폴더 위치?
A. 설치 폴더\output\ (보통 Desktop\Cotton_Print\output)
   GUI 의 [📁 output 폴더 열기] 버튼으로도 갈 수 있습니다.

Q. 같은 주문 여러 번 처리하면?
A. output 폴더에 같은 이름으로 덮어쓰여집니다.
   백업이 필요하면 이전 파일을 미리 다른 곳으로 옮겨두세요.

Q. 인터넷 안 될 때 동작?
A. BAT 가 업데이트 다운로드를 시도하다 실패하면
   자동으로 기존 버전으로 GUI 실행. 합성 기능은 동작 (단,
   주문 데이터 fetch 는 Supabase 접속이 필요해서 안 됨).
