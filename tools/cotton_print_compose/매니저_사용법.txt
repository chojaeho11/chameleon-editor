  📌 Cotton Print 인쇄 데이터 생성기 (GUI) - 매니저 사용법
═══════════════════════════════════════════════════════

작업지시서 주문번호만 입력하면 인쇄 데이터(TIFF)가
자동으로 만들어집니다. GUI 화면에서 마우스 클릭만으로 끝!


【 처음 한 번만 — 설치 】

▶ Windows 사용자

  1. 카톡으로 받은 파일들을 한 폴더에 모두 저장
     (바탕화면 또는 원하는 위치)

     필수 파일:
     - Cotton_Print_생성.bat      ← 더블클릭할 런처
     - gui.py                       (자동 다운로드 가능)
     - compose_fabric.py            (자동 다운로드 가능)

  2. Cotton_Print_생성.bat 더블클릭

  3. Python 이 없으면 자동으로 다운로드 페이지가 열림
     → "Download Python 3.x" 클릭
     → 설치 시 "Add Python to PATH" 체크박스 반드시 체크
     → 설치 완료 후 BAT 다시 더블클릭

  4. 처음 한 번은 Pillow 라이브러리가 자동 설치됨 (1-2분)

▶ Mac 사용자

  1. 받은 파일을 같은 폴더에 저장

  2. 처음에는 더블클릭이 막힘 → 우클릭 → "열기"
     → "정말 열기?" 보안 경고에서 [열기] 클릭

  3. 자동으로 GUI 가 열림


【 매번 사용 】

  1. Cotton_Print_생성  더블클릭

  2. GUI 화면이 열림:
     ┌────────────────────────────────────┐
     │  Cotton Print                       │
     │  패브릭 인쇄 데이터 생성기           │
     ├────────────────────────────────────┤
     │                                      │
     │  주문번호: [_____________]           │
     │                                      │
     │  출력 포맷: ◉ TIFF  ○ JPG  ○ PNG    │
     │                                      │
     │  항목 선택: ☑ 모든 항목              │
     │                                      │
     │  해상도: [150 ▼] dpi                 │
     │                                      │
     │  [   🎨  인쇄 데이터 생성   ]        │
     │                                      │
     └────────────────────────────────────┘

  3. 주문번호 입력 → [인쇄 데이터 생성] 버튼 클릭

  4. 5-30초 후 output 폴더가 자동으로 열림

  5. TIFF 파일을 인쇄소에 전달


【 자주 묻는 질문 】

Q. "artwork_url 없음" 메시지가 떠요
A. 작업지시서에 디자인 파일이 안 올라간 상태입니다.
   매니저 작업지시서 페이지에서 보라색 [파일 첨부] 또는
   [파일 교체] 로 업로드 후 다시 실행하세요.

Q. 파일이 너무 커요
A. 출력 포맷에서 "JPG" 선택 → 용량 1/5

Q. 인쇄 화질이 부족해요
A. 해상도에서 "300" 선택 → 화질 2배, 파일 4배 커짐

Q. GUI 가 안 열려요 / 검은 창만 잠깐 뜨고 사라져요
A. Python 이 설치 안 된 상태입니다.
   본사 IT 담당자에게 문의하거나
   https://www.python.org/downloads/ 에서 직접 설치
   (설치 시 "Add Python to PATH" 체크 필수)

Q. 처음 실행 시 시간이 오래 걸려요
A. Pillow 라이브러리 설치 + 스크립트 다운로드 중입니다 (1-2분).
   GUI 안의 로그 창에서 진행 상황이 보입니다.

Q. 결과물 폴더는 어디인가요?
A. BAT 파일이 있는 폴더의 output\ 안에 생성됩니다.
   GUI 의 [📁 output 폴더 열기] 버튼으로도 바로 갈 수 있습니다.

Q. 같은 주문을 여러 번 생성하면 어떻게 되나요?
A. output 폴더에 같은 파일명으로 덮어쓰여집니다.
   백업이 필요하면 미리 이전 파일을 다른 곳으로 옮겨두세요.
