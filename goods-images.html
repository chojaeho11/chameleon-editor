상품별 가격 편집

입력한 가격이 정상가. /goods 에서 50% 할인가 + MOQ 10 + 100개 이상 할인 으로 표시됨. 0 입력 시 "가격 문의" 모드.

💡 카드 위에 마우스 올리면 🖼️ 이미지 변경 버튼 표시. 이름·가격 input 클릭해서 직접 수정. 모든 변경은 "변경된 가격 일괄 저장" 버튼으로 한번에.

Edge Function 자동 파이프라인

supabase/functions/auto-generate-images 가 Supabase 에 배포되어 있다면, 사용자 개입 없이 자동 처리. cron 으로 매 N분마다 호출 가능 (큐 빌 때까지 반복).

📋 배포 + cron 설치 가이드

  1. Supabase CLI 로 함수 배포:
    supabase functions deploy auto-generate-images --no-verify-jwt
  2. Supabase Dashboard → Edge Functions → Secrets 에 OPENAI_API_KEY 등록 (이미 있다면 skip)
  3. 위 "한 번 호출" 버튼으로 테스트 → 성공 응답 확인
  4. cron 자동화: supabase/functions/auto-generate-images/cron.sql 파일 내용을 Dashboard → SQL Editor 에 붙여넣고 실행
  5. cron 등록 확인:
    SELECT * FROM cron.job WHERE jobname = 'auto-generate-images';

카테고리별 프롬프트 + 파일명 목록

카테고리별 "프롬프트 복사" → Midjourney/DALL-E 에서 이미지 생성 → "파일명 목록" 받아서 파일명 그대로 저장 → 탭 2 로 일괄 업로드.
